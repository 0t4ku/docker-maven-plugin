docker-maven-plugin

docker-maven-plugin

This is a Maven plugin for managing Docker images and containers for your builds.

Goals

Goal Description
docker:build Build images
docker:start Create and start containers
docker:stop Stop and destroy containers
docker:watch Watch for doing rebuilds and restarts
docker:push Push images to a registry
docker:remove Remove images from local docker host
docker:logs Show container logs
docker:logs Attach docker build archive to Maven project
